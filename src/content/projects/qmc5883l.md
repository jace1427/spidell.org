A pico-sdk library for interfacing with the QMC5883L 3-axis magnetometer. Supports reading raw x/y/z values, compass calibration, and azimuth (heading) calculation.

The calibration API lets you either run an interactive calibration routine or supply previously-known offsets directly, so subsequent builds don't need to recalibrate. See the `examples/` directory for wiring and build setup.
