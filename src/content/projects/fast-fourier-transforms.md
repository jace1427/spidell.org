A from-scratch implementation of the Fast Fourier Transform in C++, written with [Brett Sumser](https://github.com/bsumser). Applies the transform to images using the LodePNG library for PNG I/O.

Build with `make` and run with `./main`. Object files clean up with `make clean`.
