title: Building this site with Astro
description: Why Astro is a great fit for a content-focused personal site.
pubDate: 2025-02-20
tags: [astro, web]
---

I chose [Astro](https://astro.build) for this site because it ships almost no JavaScript by default, while still supporting MDX for richer blog posts when needed.

Content collections give typed frontmatter for projects and posts, and the build output is plain static HTML — easy to host on GitHub Pages with a custom subdomain.
